Fake News Detection System (Full-Stack AI Application)
Year: 2026
Tools Used: React.js, FastAPI, Python, PostgreSQL, SQLAlchemy, Redis, Docker, FAISS, Hugging Face RoBERTa, REST APIs
Live: fakenews-detector-production-6bc4.up.railway.app
- Built a full-stack fake news detection platform allowing users to submit article text or URLs and receive real-time credibility predictions with confidence scores and explanations.
- Designed a FastAPI backend with structured request and response models, integrating NLP-based classification using transformer models such as RoBERTa and lightweight fallback logic for resource-constrained deployment.
- Implemented semantic search using FAISS and embeddings to retrieve similar articles, improving contextual understanding and prediction reliability.
- Developed a React-based frontend with API integration, handling user inputs, loading states, and dynamic result visualization.
- Engineered scalable backend systems using PostgreSQL and SQLAlchemy to store articles, predictions, and user data, with Redis caching to optimize repeated queries.
- Containerized and deployed the application on Railway using Docker, resolving memory constraints by introducing a lightweight inference mode.
